# stealth-pyxel-game
Saat ini game hanya bisa di mainkan di PC/Laptop, belum bisa dimainkan di HP.

Cara main: 
1. Klik link berikut https://whitesnacc.itch.io/operation-demon-core
2. Klik "Run Game"
3. Selamat bermain!

Tombol-tombol di bagian Tutorial:
1. Klik tombol Arrow Left (<-) untuk menekan tombol '<- PREV'
1. Klik tombol Arrow Right (->) untuk menekan tombol 'NEXT ->' 
1. Klik tombol Space atau Enter untuk menekan tombol 'START'
